WELCOME!

Hello and welcome to my new blog! This is my first post on this man-page styled website.

ABOUT THIS SITE

I've redesigned my personal website using a minimalist Unix man-page aesthetic. As someone who spends a lot of time in the terminal working with genomic data, this design feels like home.

Why Man Pages?

Man pages (manual pages) are the traditional documentation format in Unix/Linux systems. They're:

  • Minimalist - Focus on content over flashy design
  • Readable - Clear typography and structure
  • Nostalgic - Reminds me of my first experiences learning Linux
  • Functional - Perfect for technical content

WHAT TO EXPECT

I'll be writing about:

  1. Bioinformatics - Genomics, transcriptomics, proteomics
  2. Computational Biology - Methods, pipelines, and tools
  3. Research Updates - Current projects and collaborations
  4. Technical Tutorials - Tips and tricks I've learned
  5. Random Thoughts - Coffee-fueled musings on science and tech

TECH STACK

This site is built with pure HTML, CSS, and JavaScript:

framework: None (pure HTML/CSS/JS)
styling: Custom CSS with CSS variables
hosting: GitHub Pages
version_control: Git
dependencies: Zero
build_process: None required

The best part? Zero dependencies and works completely offline!
