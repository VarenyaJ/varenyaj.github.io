RESEARCH PROJECTS

Fetal Anomalies Repository
Aug 2024 - Present | Columbia University & Broad Institute

Developing genotype-phenotype databases for rare diseases. Training local large language models to extract Human Phenotype Ontology (HPO) terms from prenatal ultrasound data and creating federated phenopacket repositories.

Technologies: Python LLMs HPO Phenopackets

Reinforcement Learning for UAV Relay Positioning
Coordinated Science Laboratory, UIUC

Conducted research on implementing novel reinforcement learning algorithms for autonomous UAV positioning to optimize communication relay networks.

Technologies: Python Reinforcement Learning UAV Systems Optimization

NExt-generation Sequencing Analysis Toolkit (NEAT)
National Center for Supercomputing Applications (NCSA)

Implemented parallel processing optimizations for genomic analysis pipelines. Integrated DRAGEN enrichment-based CNV detection into clinical variant interpretation workflows.

Technologies: Python GATK DRAGEN Parallel Computing HPC

Low-Coverage WGS for Phylogenetic Markers
Illinois Natural History Survey

Validated novel pipeline for using Low-Coverage Whole Genome Sequencing as a cost-efficient method for assembling phylogenetic markers in evolutionary biology studies.

Technologies: WGS Bioinformatics Phylogenetics Pipeline Development
